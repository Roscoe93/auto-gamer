Cross-platform operator console

Five UI directions for the game automation desktop app.

Each prototype keeps the same product core in view: window selection, script orchestration, live logs, recognition preview, and stop controls. What changes is the personality, pacing, and operational mood.

ScopeWindows + macOS
TargetWindowed desktop games
FormatStatic HTML prototypes
FocusLogs, start/stop, extensible scripts

Nocturne Operator

Poetic, quiet, and cinematic. Reads like a control room for long overnight runs with soft contrast and calm status rhythm.

Editorial / atmospheric / reflective

Signal Workshop

Industrial and practical. Built for dense scanning, quick toggles, and a little field-manual toughness without feeling messy.

Utilitarian / bright / workshop

Neon Tactical

High-contrast command center energy. More kinetic, more game-adjacent, better for teams who like immediate visual response.

Tactical / high-signal / vivid

KuroNeko Studio

Softer and more humane. Good when the operator spends long hours with the tool and wants less visual fatigue without losing precision.

Organic / calm / low fatigue / selected

Telemetry Brut

Bold, raw, and unapologetic. Prioritizes commanding hierarchy and operational pressure over polish in the conventional sense.

Brutalist / assertive / command-heavy
